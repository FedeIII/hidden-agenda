# Hidden Agenda
An abstract board game with psychology

## Getting Started

### Installing
* Install [yarn](https://yarnpkg.com/en/docs/install)
* Install dependencies
```
yarn install
```
* Build dist folder
```
yarn do
```
* Open index.html in the dist folder

### Developer
Watch for changes to build again the dist folder
```
yarn go
```
